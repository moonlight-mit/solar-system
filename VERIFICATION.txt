TROUBLESHOOTING QUICK REFERENCE
================================================================================

IF PYTHON NOT FOUND:
  → Install from python.org
  → Check environment variables
  → Restart terminal

IF PORT 8000 IN USE:
  → Kill process: lsof -i :8000 (Mac/Linux)
  → Use different port: Change in app.py
  → Wait 30 seconds and retry

IF API CALLS FAIL:
  → Verify backend running: http://localhost:8000
  → Check browser console: F12
  → Check server terminal for errors
  → Verify CORS enabled in app.py

IF CALCULATIONS WRONG:
  → Click "♻️ Reset Default"
  → Verify control panel values
  → Check against NASA data
  → Clear browser cache

IF FRONTEND DOESN'T LOAD:
  → Try: http://localhost:8000/templates/index.html
  → Or: file:///d:/tryyyyyyyyy%20solar2copy/templates/index.html
  → Refresh browser (F5)
  → Check Three.js CDN available
