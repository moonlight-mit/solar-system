================================================================================
❓ TROUBLESHOOTING
================================================================================

Problem: "Python not found"
Solution: Install Python 3.8+ from python.org

Problem: Port 8000 already in use
Solution: 
  lsof -i :8000  (check what's using it)
  Change port in app.py: uvicorn.run(..., port=8001)

Problem: API calls fail
Solution:
  - Check backend is running (you should see "Uvicorn running")
  - Check browser console for actual error
  - Verify URL in solar_api.js

Problem: Physics calculations seem wrong
Solution:
  - Reset to default values (click "Reset" in control panel)
  - Check sun mass and gravity factors
  - Verify against known planet values in README
