HOW TO USE THE PHYSICS LAB
═══════════════════════════════════════════════════════════════════════════════

1. OPEN PHYSICS LAB
   Click: "🔭 Vật Lý" button in main menu
   
2. USE CALCULATOR 1: Orbital Velocity
   - Select planet (Mercury to Saturn)
   - Click "⚡ Tính tốc độ"
   - See velocity in km/s
   
   Example: Earth = 29.79 km/s

3. USE CALCULATOR 2: Gravitational Force
   - Select two bodies (Sun, planets, moon)
   - Set distance in AU
   - Click "💥 Tính lực"
   - See force in scientific notation
   
   Example: Sun-Earth = 3.54e22 N

4. USE CALCULATOR 3: Escape Velocity
   - Select planet/star
   - Click "🚀 Tính toán"
   - See escape velocity
   
   Example: Earth = 11.19 km/s

5. MODIFY CONTROL PANEL
   - Change sun mass → Velocities change
   - Adjust gravity → Forces recalculate
   - Control speed → Watch animation
   - Click "Apply" to recalculate all

KEY PHYSICS FORMULAS
═══════════════════════════════════════════════════════════════════════════════

Orbital Velocity:
  v = √(GM/r)
  
  Example: Earth at 1 AU:
  v = √((6.674×10⁻¹¹ × 1.989×10³⁰) / 1.496×10¹¹)
  v ≈ 29,785 m/s = 29.79 km/s ✓

Escape Velocity:
  v_e = √(2GM/r)
  
  Example: Earth surface:
  v_e = √(2 × 6.674×10⁻¹¹ × 5.972×10²⁴ / 6.371×10⁶)
  v_e ≈ 11,186 m/s = 11.19 km/s ✓

Gravitational Force:
  F = G·m₁·m₂/r²
  
  Example: Sun-Earth:
  F = 6.674×10⁻¹¹ × 1.989×10³⁰ × 5.972×10²⁴ / (1.496×10¹¹)²
  F ≈ 3.54 × 10²² N ✓

Kepler's Third Law:
  T² = (4π²/GM) × a³
  
  Means: Orbital period squared ∝ semi-major axis cubed
